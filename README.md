## Rules

1. **Players:** The game is played between two players. 

2. **Rolling the Dice:** Each player takes turns rolling two six-sided dice. The sum of the two dice is calculated for each player.

3. **Winning Conditions:**
   - If a player rolls a sum of 2 (both dice showing 1), they win immediately.
   - Otherwise, the player with the highest sum wins.
   - In case of a tie, it's declared as a tie game.
